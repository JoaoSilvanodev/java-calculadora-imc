# Calculadora de IMC (JavaFX)

## Descrição

Aplicação desktop desenvolvida em Java utilizando JavaFX para cálculo do Índice de Massa Corporal (IMC).
O sistema permite ao usuário inserir peso e altura, realizando o cálculo automaticamente e exibindo a classificação de acordo com padrões de saúde.

---

## Funcionalidades

* Cálculo automático do IMC
* Classificação do resultado (abaixo do peso, normal, sobrepeso, etc.)
* Aceita altura em metros ou centímetros (ex: 1.78 ou 178)
* Suporte a números com vírgula ou ponto
* Tratamento de entradas inválidas

---

## Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Separação de responsabilidades (uso de métodos para lógica)
* Tratamento de exceções (try/catch)
* Manipulação de entrada do usuário
* Interface gráfica com JavaFX

---

## Tecnologias utilizadas

* Java
* JavaFX

---

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/java-calculadora-imc.git
```

2. Abra o projeto em uma IDE (IntelliJ IDEA recomendado)

3. Execute a classe principal:

```java
CalculadoraIMC.java
```
